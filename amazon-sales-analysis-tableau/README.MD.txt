# 📊 Amazon Sales Analysis Dashboard (Tableau)

## Project Overview
This project presents an Tableau dashboard built to analyze Amazon sales performance across states, product categories, shipping status, and weekly trends.

## Key Insights
- State-wise sales quantity distribution across India
- Impact of shipping service level (Expedited vs Standard)
- Order status breakdown (Shipped, Cancelled, Returned)
- Weekly sales and revenue trends by category
- Product size and category contribution analysis

## Tools Used
- Tableau
- Data Visualization
- Sales Analytics
- Business Intelligence
