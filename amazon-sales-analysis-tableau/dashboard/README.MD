## Amazon Sales Data Analysis Dashboard
## Project Overview
This project presents an interactive sales analytics dashboard developed to analyze order quantity, sales performance, and logistics insights from an e-commerce dataset. The dashboard highlights key metrics such as total products sold, total revenue, shipping performance, category demand, and regional distribution of orders.

The objective of this project is to extract meaningful business insights that can help stakeholders understand sales trends, customer demand, product performance, and operational efficiency
